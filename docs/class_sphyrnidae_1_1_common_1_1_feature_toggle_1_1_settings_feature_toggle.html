static

Retrieves a feature toggle value as a boolean - use this if you are doing multiple lookups to avoid doing ServiceLocation each time on the service/cache

All of the following values will return true (case insensitive): TRUE T YES Y ON 1 CHECK CHECKED

All of the following values will return false (case insensitive): FALSE F NO N OFF 0 UNCHECK UNCHECKED

Anything else will also return the default value

Parameters
servicesThe services needed for the actual lookup
nameThe name of the feature toggle to retrieve
defaultValueIf the feature toggle is not found in the collection from IFeatureToggle.GetAll(), this will be returned instead Default: false
Returns
The bool value of the feature toggle
